# Руководство пользователя: GigaAM & PyAnnote Telegram Transcriber

Этот бот предназначен для транскрибации (перевода речи в текст) аудиофайлов с использованием передовых моделей искусственного интеллекта. Он поддерживает разделение по спикерам (диаризацию) и работу с длинными записями.

## Основные возможности

*   **Распознавание речи**: Высокая точность для русского языка благодаря модели GigaAM v3.
*   **Разделение спикеров**: Автоматическое определение разных голосов в записи.
*   **Длинные файлы**: Поддержка записей длительностью несколько часов.
*   **Форматы**: Поддержка большинства аудиоформатов (MP3, WAV, OGG, M4A и др.) и голосовых сообщений Telegram.

## Начало работы

1.  **Запуск**: Нажмите кнопку `Start` или отправьте команду `/start` боту.
2.  **Отправка файла**:
    *   Просто перетащите аудиофайл в чат с ботом.
    *   Или перешлите голосовое сообщение из другого чата.
3.  **Ожидание**:
    *   Бот добавит ваш файл в очередь.
    *   Вы получите уведомление, когда начнется обработка.
    *   По завершении бот пришлет текстовый файл с расшифровкой.

## Команды бота

*   `/start` — Приветственное сообщение и проверка работоспособности.
*   `/help` — Показать эту справку.
*   `/status` — Узнать позицию вашего файла в очереди.
*   `/cancel` — Отменить обработку текущего файла.

## Формат результата

Результат предоставляется в виде текстового файла (`.txt`).
Пример содержимого:

```text
[00:00:05 - 00:00:12] SPEAKER_00: Здравствуйте, это начало записи.
[00:00:13 - 00:00:18] SPEAKER_01: Я понимаю вашу точку зрения.
[01:45:30 - 01:45:45] SPEAKER_00: Это работает и для длинных записей.
```

*   **Временные метки**: Показывают начало и конец фразы.
*   **SPEAKER_XX**: Условное обозначение спикера (например, SPEAKER_00, SPEAKER_01). Бот не знает реальных имен, но различает голоса.

## Частые вопросы (FAQ)

**В: Сколько времени занимает обработка?**
О: Это зависит от длины записи и загруженности сервера. Обычно обработка занимает от 1/5 до 1/2 времени звучания записи (например, час аудио обрабатывается 10-30 минут).

**В: Есть ли ограничения?**
О: Да, максимальная длина файла — 3 часа. Максимальный размер файла — 500 МБ. Вы можете отправить не более 3 файлов одновременно.

**В: Бот пишет "Очередь переполнена". Что делать?**
О: Подождите немного. Сервер обрабатывает много запросов. Попробуйте отправить файл через 10-15 минут.

**В: Почему не распознались спикеры?**
О: Если запись очень короткая или голоса очень похожи/зашумлены, модель может не различить спикеров. В этом случае весь текст будет приписан одному спикеру.
